Useful Videos And Links

Here is a list of videos and links that have greatly shaped me as a developer and as a person.

  • Scale Yourself by Scott Hanselman

    • This videos actually lead me to create a blog. It talks about how to set yourself up so you can do more.

  • Full Stack Python

    • This is an open online book about using python's Flask to build a full application. Even though I don't always use Flask this book made me understand certain concepts in frontend, backend, deployments and scaling of web applciations.

  • Continous Delivry and Pipelines

    • This video helped me understand the importance of build pipelines and the most current technology that the industry is moving towards to be able to build faster and more reliably.
